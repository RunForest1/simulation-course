### Вывод 

В ходе выполнения лабораторной работы был реализован алгоритм генерации дискретной случайной величины методом интервалов и генератор нормальной случайной величины методом Бокса-Мюллера. Проведена серия экспериментов при различных объемах выборки ($N = 10, 100, 1000, 10000$).

**Основные выводы:**

1. **Метод интервалов** корректно преобразует равномерную случайную величину $\alpha \in [0,1)$ в дискретную с заданным распределением вероятностей.

2. **Метод Бокса-Мюллера** успешно генерирует нормальную случайную величину $N(0, 1)$: выборочные характеристики сходятся к теоретическим ($\mu = 0$, $\sigma^2 = 1$) при росте $N$.

3. **Закон больших чисел подтвержден для обеих моделей:** 
   - При малых $N$ (10, 100) наблюдаются значительные погрешности эмпирических характеристик (до 50% и выше)
   - При больших $N$ (1000, 10000) погрешности стремятся к нулю (<1%), выборочные среднее и дисперсия сходятся к теоретическим значениям

4. **Гистограммы нормального распределения** наглядно демонстрируют сходимость:
   - При $N = 10, 100$ форма гистограммы нерегулярна и отличается от теоретической кривой Гаусса
   - При $N = 1000, 10000$ гистограмма приобретает характерную симметричную «колоколообразную» форму, соответствующую плотности нормального распределения

5. **Критерий $\chi^2$** подтвердил работоспособность обоих генераторов:
   - Для ДСВ: статистика $\hat{X}^2 = \sum \frac{(n_i - N p_i)^2}{N p_i}$ при больших $N$ не превышает $\chi^2_{крит} = 9.488$ ($\alpha=0.05$, $k=4$)
   - Для нормальной величины: гипотеза о нормальности принимается при $N \ge 1000$, что согласуется с теоретическими ожиданиями
