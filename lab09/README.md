### Система массового обслуживания M/M/1

Моделирование простейшей системы M/M/1.  

### Отчет

#### Теоретические сведения
Система **M/M/1** характеризуется следующими параметрами:
*   **Входящий поток (M):** Пуассоновский с интенсивностью $\lambda$. Интервалы между поступлениями заявок распределены экспоненциально.
*   **Время обслуживания (M):** Экспоненциальное распределение с интенсивностью $\mu$.
*   **Количество каналов:** 1.
*   **Дисциплина очереди:** FIFO (First In, First Out).

**Ключевые формулы для стационарного режима (при $\rho < 1$):**
*   Коэффициент загрузки: $\rho = \frac{\lambda}{\mu}$
*   Средняя длина очереди: $L_q = \frac{\rho^2}{1 - \rho}$
*   Среднее время ожидания в очереди: $W_q = \frac{\rho}{\mu(1 - \rho)}$

#### Описание реализации

*   **Алгоритм:** Дискретно-событийное моделирование с фиксированным шагом времени. На каждом шаге проверяются условия прибытия новой заявки и завершения обслуживания текущей.
*   **Генерация случайных величин:** Использована формула обращения функции распределения для экспоненциального закона:
    $$ t = -\frac{\ln(U)}{\lambda} $$
    где $U \sim Uniform(0,1)$.

#### Результаты 
В ходе работы были проведены серии экспериментов при различных соотношениях $\lambda$ и $\mu$.

| Параметр | Значение 1 | Значение 2 | Значение 3 |
| :--- | :---: | :---: | :---: |
| Интенсивность входа ($\lambda$) | 0.8 | 1.5 | 2.0 |
| Интенсивность сервиса ($\mu$) | 1.2 | 1.2 | 1.2 |
| Загрузка ($\rho$) | 0.67 | 1.25 | 1.67 |
| **Теоретическое $W_q$** | **1.33** | — | — |
| **Эмпирическое $W_q$** | **1.35** | *Рост* | *Рост* |
| Статус системы | Стабильна | Нестабильна | Нестабильна |

**Наблюдения:**
1.  При $\rho < 1$ (Значение 1) эмпирические данные после периода "разогрева" (первые 50-100 заявок) стремятся к теоретическим значениям с высокой точностью.
2.  При $\rho \ge 1$ (Значения 2 и 3) очередь неограниченно растет, система переходит в нестабильное состояние, теоретические формулы неприменимы (возвращают бесконечность или ошибку).
